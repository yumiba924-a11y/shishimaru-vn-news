# VN30 図解新聞 — 構成 v2（寄与度ドリブン）
更新: 2026/6/30
位置づけ: リリース版v1（静止版・運用中）の上に重ねる「中身の深化」。
この構成で Claude Design に動く版を作らせ、Claude Code に量産テンプレ化させる。

## v1 → v2 の変更点（一言で）
- v1=「何が起きたか」を数字で見せる（タイル＋セクター）。
- v2=「**誰が指数を動かしたか**」を寄与度で見せる。指数チャート/スパークラインを捨て、寄与度を主役に。

## 1. 構成（上から）
```
①ヘッダー：日付＋「昨日のベトナム相場」
②指数バー：VN-Index と VN30 の終値・騰落率のみ（※チャート/スパークラインは廃止）
③昨日のひとこと：大見出し15字以内 ＋ 内容60字以内（米国株図解新聞と同型）
④寄与度（新・主役）：誰が指数を何pt押し上げ/押し下げたか＝「綱引きの収支」
⑤VN30タイル30銘柄（騰落率順）＋ セクター別強弱バー
⑥テーマ別カード：急変銘柄が出た日だけ、CafeF等で理由を確認し自分の言葉で1-2行
⑦マクロ帯：ドン/円・USD/VND・金・原油
```

## 2. 各ブロックの作り方（データ仕様）
### ④寄与度（v2の核）
- 計算: contrib_i(％pt) = weight_i × pct_i。weight_i = close_i×発行済株式数 ÷ VN30合計時価総額。
- 発行済株式数は `config\vn30_universe.json` に固定値で保持（増資時のみ手更新）。
- **必ず「概算寄与度」と明記**（単純時価総額ウェイト。正式VN30＝浮動株調整・外国人枠キャップ後とはズレる）。
- 見せ方=「綱引きの収支」：押し上げ合計 +X.Xpt ／ 押し下げ合計 -X.Xpt ＋ 上位の押上/押下銘柄。
- 6/29実証値: VIC -1.3pt / VHM -0.4pt（不動産計 -1.7pt）vs 銀行13社 +0.5pt → 「13社の銀行 vs 2社のVingroup、Vingroupが勝って小幅安」。
- ※ネット概算と実際の指数騰落はズレる（VICのウェイトが正式より大きく出る）。相対の大小で見せれば十分。

### ③昨日のひとこと（米国版と同型）
- 大見出し: 全角15字以内（例「VIC急落、銀行も支えきれず」14字）
- 内容: 全角60字以内（例「VICだけで指数を約1.3pt押し下げ。銀行13社の上昇(+0.5pt)も及ばず小幅安。物色は不動産から銀行へ。」56字）
- ④寄与度から機械生成 → 弓場さんが1回通す（半自動）。

### ⑥テーマ別カード（急変時のみ・反復回避）
- トリガー: |騰落率| ≧ 閾値（既定 ±3%、config `card_threshold_pct`）の銘柄が出た日だけ発火。
- 中身: CafeF（cafef.vn）等で急変理由を確認 → **自分の言葉で1-2行に要約**（記事転載不可）。
- 動きが小さい日はカード無し（数字だけ）→ 毎日ネタを探さない＝続く。
- カードは上位6本まで表示（残りは件数のみ）。理由欄は空＝人が後で埋める半自動。

## 3. ニュースソース（③⑥用・確定）
| ソース | 用途 |
|---|---|
| CafeF (cafef.vn) | 急変理由・外国人動向(khối ngoại)・市場概況 |
| VnExpress International | 英語の概況補完 |
| The Investor (theinvestor.vn) | 機関投資家目線 |
- 「読んで事実を自分の言葉で1-2行」。原文転載しない。
- CafeFは寄与度計算と同じ構造（Vingroup圧力で指数安）を言葉で裏付ける＝データと記事の整合が取れる。

## 4. Claude Design への依頼（動く版テンプレ）
- v1のHTML＋このv2構成を渡す。CQC Design System適用。
- アニメ:
  1. 指数の数字がカウントアップ
  2. **寄与度の「綱引き」バーが中央から左右に伸びる**（v2の主役なので一番見せる）
  3. VN30タイルが騰落率順に上から順次フェードイン
  4. セクターバーが中央軸から左右に伸びる
- 品位ある動き（ease-out / 各0.3-0.6秒 / 全体2-3秒）。
- **毎日変わる数値はベタ書きせず、データ部分とテンプレート部分を分離**（後でCodeが `interpreted.json` から差し込む）。

## 5. Claude Code への依頼（量産化）※本実装で完了済みの範囲
- [x] ④寄与度計算を interpret 層に実装（`interpretation.contribution`）。
- [x] 発行済株式数を config 化（`constituents[].shares`、VNDirect OUTSTANDING_SHARES由来）。
- [x] ②大見出し15字＋内容60字を寄与度から自動生成。
- [x] ⑥カードのトリガー（±3%超）。理由テキストは半自動（空欄＝人が1-2行）。
- [x] ②指数バーからチャート/スパークライン撤去。
- [ ] Designの動く版HTMLが出たら `render` に組み込み、`interpreted.json` から量産化（次フェーズ）。

## 6. 留意（正直な弱点）
- 寄与度は概算（正式VN30ウェイトではない）。「概算」明記必須。精度を上げるなら正式構成比に差し替え。
- CafeF等の本文は著作権物。要約のみ。転載不可。
- ⑥のニュース要約は最後は人の手（半自動）。全自動にしない。
