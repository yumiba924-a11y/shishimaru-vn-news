# ししまるベトナム新聞 / ししまるVN30分解新聞

ベトナム株（VN30）の「図解新聞」を、無料データ＋現地報道の要約から作るプロジェクト。
専用端末は不要。米国の人気アカウント「ぶたまる🐽米国株投資」の図解新聞をベンチマークに、ベトナム版を作る。

**ライブ（レイアウト調整用）** → https://yumiba924-a11y.github.io/shishimaru-vn-news/

## 2媒体構成
| | デイリー | ウィークリー |
|---|---|---|
| 媒体 | **ししまるベトナム新聞** | **ししまるVN30分解新聞** |
| 対象 | ベトナム市場全体 | VN30の30銘柄 |
| 主役 | ニュース（現地報道の要約）＋ヒートマップ | 寄与度・セクター分解 |
| 頻度 | 毎営業日の朝 | 週1 |

思想：**「言葉で分解し、図解で裏付ける」**。言葉が主、図解（ヒートマップ・数字・横棒）が従。

## 構成
```
docs/index.html     現行レイアウト（ぶたまる型・手調整中／GitHub Pagesで公開）
docs/assets/        ししまる画像（マスコット）
base/               レイアウトの素HTML（Design抽出版）
scripts/            データパイプライン（取得→解釈→描画）
config/             VN30構成30銘柄＋セクター＋発行済株式数
outputs/            生成データ・図解（日付別）
```

## データソース
| 用途 | ソース | 状態 |
|---|---|---|
| VN30 30銘柄 終値・騰落率・出来高 | VNDirect finfo | 無料・キー不要 |
| VN-Index / VN30 指数 | VNDirect dchart | 無料・キー不要 |
| 発行済株式数（寄与度の概算ウェイト用） | VNDirect → config固定値 | 年単位で更新 |
| 為替 USD/VND・ドン/円 | open.er-api.com | 無料・ライブ |
| 金 | gold-api.com | 無料・ライブ |
| ニュース（市況コメント） | CafeF / VnExpress / The Investor / Vietstock 等 | 現地報道・**要約のみ／転載不可** |

## ニュースの作り方（差別化の核・半自動）
- 現地報道を読み、**事実を自分の言葉で1〜2行に要約**（原文転載は不可）。専門用語は平易に。
- ミクロ（個別株・セクター）とマクロ（政策・ドン相場・外国人フロー等）を両方。
- ±3%超の急変があった日は、その理由を優先してカード化。
- 要約は最後は人（or Claude）の手が入る半自動。全自動にしない。

## 寄与度（週次の主役）
- contrib ＝ ウェイト × 騰落率。ウェイト ＝ 終値 × 発行済株式数 ÷ VN30合計。
- 単純時価総額ウェイトのため **「概算」** と明記する（正式VN30ウェイト＝浮動株調整後とはズレる）。
